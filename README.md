# My Nvim Config

## Getting Started

Clone the repo
```bash
# On Linux
git clone https://github.com/SushiWaUmai/my-nvim-config.git ~/.config/nvim

# On Windows
git clone https://github.com/SushiWaUmai/my-nvim-config.git %localappdata%/nvim
```

Use Alpha or 
```vim
:PackerSync
```
to install all plugins.

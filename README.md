## Getting Started

Clone the repo
```bash
# On Linux
git clone https://github.com/SushiWaUmai/nvim.git ~/.config/nvim

# On Windows
git clone https://github.com/SushiWaUmai/nvim.git %localappdata%/nvim
```

Use Alpha or 
```vim
:Lazy
```

to install all plugins.
